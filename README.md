# Eye EMR — ระบบเวชระเบียน Eye Clinic

ระบบ EMR สำหรับคลินิกขนาดเล็ก ใช้งานในเครือข่ายภายใน (LAN) เท่านั้น

## ฟีเจอร์

- **คลินิก** — รายชื่อผู้ป่วยรายวัน พร้อมสถานะ (รอตรวจ → กำลังตรวจ → เสร็จสิ้น) และเปิด visit ใหม่
- **ผู้ป่วย** — ค้นหาด้วย HN/ชื่อ/เบอร์โทร, ประวัติการรักษาทั้งหมด, แถบเตือนแพ้ยา
- **Visit** — บันทึก vitals, อาการ, การตรวจ, วินิจฉัย; สั่งยาจากสต็อก (คำนวณราคาอัตโนมัติ) + หัตถการ/อุปกรณ์
- **ปิด visit** — บันทึกการชำระเงิน หักสต็อกยา และพิมพ์ใบเสร็จ (A5) ในขั้นตอนเดียว; เปิด visit ซ้ำได้ภายในวันเดียวกัน
- **พิมพ์เอกสาร** — ใบเสร็จ / ใบนัด / ฉลากยา (สติกเกอร์ A5) พิมพ์คลิกเดียว และพิมพ์ทั้งหมดอัตโนมัติเมื่อปิด visit ได้
- **สต็อกยา** — จัดการรายการยา รับเข้า/ปรับออก แจ้งเตือนยาใกล้หมด + หัตถการ/อุปกรณ์ (เพิ่มอัตโนมัติทุก visit ได้ เช่น ค่าตรวจ)
- **รายงาน** — สรุปรายรับรายวัน/รายเดือน
- HN รันอัตโนมัติรูปแบบ `69-0001` (ปี พ.ศ. + เลขลำดับ รีเซ็ตทุกปี)

ดูบันทึกการเปลี่ยนแปลง/รายละเอียดเชิงเทคนิคได้ที่ [`docs/CHANGES.md`](docs/CHANGES.md)

## การติดตั้ง

```bash
npm install
npm run db:generate   # สร้าง migration จาก schema (ครั้งแรก / เมื่อ schema เปลี่ยน)
npm run seed          # สร้างผู้ใช้เริ่มต้น + ยาตัวอย่าง
npm run dev           # โหมดพัฒนา
```

ผู้ใช้เริ่มต้น: `eye` / `eyeclinic` — เปลี่ยนรหัสผ่านได้ในหน้าตั้งค่า

## ใช้งานจริงบนเครื่องในคลินิก

```bash
npm run build
npm start             # เปิดที่ http://<ip-เครื่อง>:3000
```

ฐานข้อมูลทั้งหมดอยู่ในไฟล์เดียว: `data/clinic.db` — **สำรองข้อมูลโดยก๊อปปี้ไฟล์นี้เป็นประจำ**
(แนะนำ: ตั้ง Task Scheduler/cron ก๊อปปี้ไป external drive ทุกวัน)
